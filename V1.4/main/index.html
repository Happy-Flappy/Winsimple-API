    Namespace: ws (WinAPI Converters)

    Utility functions for converting between Windows API types and C++ standard types.

    Functions

    bool ResolveRelativePath(std::string path)

    Converts a relative path to absolute path using the executable's directory. - Parameters: path - The relative or absolute file path - Returns: true if file exists, false otherwise - Note: Updates the path parameter to absolute path

    std::wstring WIDE(std::string str)

    Converts UTF-8 string to wide string (UTF-16). - Parameters: str - UTF-8 encoded string - Returns: Wide character string

    std::string SHORT(const std::wstring& wstr)

    Converts wide string (UTF-16) to UTF-8 string. - Parameters: wstr - Wide character string - Returns: UTF-8 encoded string

    LPCSTR TO_LPCSTR(std::string str)

    Converts std::string to LPCSTR (C-style string). - Parameters: str - Standard string - Returns: C-style string pointer

    std::wstring GetShortPathNameSafe(const std::wstring& longPath)

    Converts long Windows path to short path format (8.3). - Parameters: longPath - Long path name - Returns: Short path name or empty string on failure - Usage: Required for MCI functions that don't support long paths

    Namespace: ws (Data Types)

    Template-based vector and rectangle structures with automatic type conversion.

    Vector Types

    struct Vec2i

    2D integer vector with type conversion support. - Members: int x, y - Constructors: - Vec2i() - Default constructor - Vec2i(T x_val, U y_val) - Template constructor for any arithmetic types - Vec2i(const T& other) - Constructor from any type with .x, .y members - Conversion Operators: - operator T() const - Convert to any type with .x, .y members - operator POINT*() - Convert to Windows POINT pointer - operator const POINT*() const - Const POINT pointer conversion

    struct Vec2f

    2D float vector with identical interface to Vec2i (float members).

    struct Vec2d

    2D double vector with identical interface to Vec2i (double members).

    struct Vec3i

    3D integer vector with type conversion support. - Members: int x, y, z - Constructors: Support for three arithmetic types or types with .x, .y, .z members

    struct Vec3f

    3D float vector with identical interface to Vec3i.

    struct Vec3d

    3D double vector with identical interface to Vec3i.

    Rectangle Types

    struct IntRect

    Integer rectangle with width/height style. - Members: int left, top, width, height - Constructors: - IntRect(T1 l, T2 t, T3 w, T4 h) - Template constructor for any arithmetic types - IntRect(const T& other) - Constructor from any rect-like type (supports both width/height and right/bottom styles) - Conversion Operators: - operator T() const - Convert to any rect-like type

    struct FloatRect

    Float rectangle with identical interface to IntRect (float members).

    struct DoubleRect

    Double rectangle with identical interface to IntRect (double members).

    Note: All rectangle types automatically handle conversion between: - Width/height style (e.g., sf::IntRect, custom types with .left, .top, .width, .height) - Right/bottom style (e.g., WindowsRECTwith .left, .top, .right, .bottom)

    Namespace: ws (Sound and Video)

    Audio playback using Windows MCI (Media Control Interface).

    Class: Wav

    Manages audio file playback with channel-based system.

    Public Members

    std::string path    // File path
int channel         // Audio channel ID (0-100)
bool blocking       // Whether playback blocks other operations
std::string ID      // Internal channel identifier
std::string extension // File extension

    Constructor & Destructor

    Wav(std::string path = "", int channel = 0, bool blocking = true)
~Wav() // Automatically closes MCI channel

    Static Methods

    static int getFreeChannel()

    Finds an available audio channel (0-100). - Returns: Available channel ID or -1 if none available - Note: Channels are considered free if not "playing" or "paused"

    static std::string getChannelStatus(int m_channel)

    Gets the current status of a channel. - Parameters: m_channel - Channel ID to check - Returns: Status string ("playing", "stopped", "paused", "error")

    static bool PlayFree(std::string m_path, int m_channel, bool m_blocking = false)

    Plays an audio file on the specified channel. - Parameters: - m_path - Audio file path - m_channel - Channel ID - m_blocking - Whether to block during playback - Returns: true on success, false on failure - Note: Automatically converts long paths to short paths

    static void isSupported(std::string m_path)

    Checks if file format is supported and issues warnings. - Always Supported: .wav, .mid, .midi - Usually Supported: .mp3, .wma - Limited Support: .au, .aif, .aiff, .snd

    Instance Methods

    bool open(std::string m_path, int m_channel, bool m_blocking = true)

    Opens an audio file for playback. - Returns: true on success - Note: Sets up MCI device and time format in milliseconds

    void play()

    Starts playback on the opened channel. - Behavior: Respects blocking setting; won't restart if already playing and blocking

    void stop() / static void stop(int m_channel)

    Pauses playback on the channel.

    bool setVolume(int percent)

    Sets volume level (0-100%). - Returns: true on success - Note: Not supported for MIDI files; uses MCI volume range 0-1000

    int getVolume()

    Gets current volume level (0-100%). - Returns: Volume percentage or 0 on error

    bool setProgress(long seconds)

    Seeks to specified position in seconds. - Returns: true on success - Behavior: Different commands for playing vs paused/stopped states

    long getProgress()

    Gets current playback position in seconds. - Returns: Position in seconds or 0 on error

    long getLength()

    Gets total duration in seconds. - Returns: Duration in seconds or 0 on error

    bool isFinished()

    Checks if playback has completed. - Returns: true if progress >= length

    Usage Example

    #include "winsimple.hpp"

int main() {
    ws::Window window(800, 600, "My Application");
    ws::Texture playerTexture("player.png");
    ws::Sprite player;
    player.setTexture(playerTexture);
    player.setPosition(100, 100);

    ws::Timer timer;

    while(window.isOpen()) {
        MSG msg;
        while(window.pollEvent(msg)) {
            // Handle events
        }

        // Update
        if(ws::Global::getKey('W')) player.y -= 5;
        if(ws::Global::getKey('S')) player.y += 5;

        // Render
        window.clear();
        window.draw(player);
        window.display();
    }

    return 0;
}

    Thread Safety

    • Not Thread-Safe: All GUI operations must be performed on the main thread
    • Window Messages: Must be processed on the thread that created the window
    • GDI+ Objects: Should not be shared across threads

    Memory Management

    • RAII: Most classes use RAII for automatic resource cleanup
    • Move Semantics: Texture and other resource-heavy classes support move semantics
    • No Copying: Window class is non-copyable to prevent handle duplication issues

    Error Handling

    • File Operations: Print errors to stderr and return false/nullptr on failure
    • Windows API: Check GetLastError() for detailed error information
    • MCI Errors: Automatically decoded to human-readable strings

    Dependencies

    • Windows XP or later (Can support down to windows 95 if GDI+ is provided manually)
    • GDI+ (ships with Windows XP and later)
    • Standard C++ library
    • Windows Common Controls library
